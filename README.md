# CashoutSPayLater.my

Landing page untuk perkhidmatan pencairan limit kredit (ShopeePay Later, dll). Dibangunkan dengan Next.js, TypeScript, dan Tailwind CSS.

## Tech Stack
- **Framework:** Next.js 14+ (App Router)
- **Styling:** Tailwind CSS (Modern Setup) + CSS Variables
- **Language:** TypeScript
- **Icons:** Lucide React
